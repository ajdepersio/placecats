Placecats is a simple to use image placeholder webservice for developers featuring everyone's favorite domesticated animal. Images can be requested for a common aspect ratio, or at a specific resolution. Or, if you just want a cat and you don't care what size, we can do that too.

Future plans include the ability to request cats of a general size and with a specific color tint applied.

I hope to make Placecats the premier image placeholder service, or at the very least make the development process a bit more fun.

Endpoint Description Example
http://cats.depersio.net/random Returns a random cat image of a random aspect ratio. Meow!
http://cats.depersio.net/square Returns a random cat image of a 1-to-1 aspect ratio. Meow!
http://cats.depersio.net/standard Returns a random cat image of a 4-to-3 aspect ratio. Meow!
http://cats.depersio.net/wide Returns a random cat image of a 16-to-9 aspect ratio. Meow!
http://cats.depersio.net/tall Returns a random cat image of a 9-to-16 aspect ratio. Meow!
http://cats.depersio.net/panorama Returns a random cat image of a 3-to-1 aspect ratio. Meow!
http://cats.depersio.net/:width/:height Returns a random cat image of the specified size. Meow!

We all love cats, but unfortunately not all cats have the life they deserve. If you love this service, I hope you can find it in your heart to give back somehow.

One way you can do so is by contributing to Placecats pictures of your cat! Making the service better can only encourage more people to use it and raise awareness for cats. Not just any cat picture will do, visit the following link for more information.

If you're a programmer, you could also contribute to the Placecats source code here. It's not very complex, a good way to start your open source career.

If you wish to make monetary donations, you are free to do so to the charity of your choice.

Alternatively you can donate to Placecats. Once the hosting costs ($30 yearly) are covered, 100% of donations received will go to the Humane Society of Indianapolis.
